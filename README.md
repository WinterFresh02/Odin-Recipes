# Odin Recipes

## Introduction

Odin Recipes is a beginner-level web development project completed as part of The Odin Project curriculum. The purpose of this project is to create a simple recipe website using only HTML while practicing the fundamentals of web page structure and navigation.

## Skills Demonstrated

By completing this project, I have demonstrated:

* Understanding of basic HTML document structure
* Use of semantic HTML elements
* Creating headings, paragraphs, and lists
* Linking pages using relative links
* Organizing content across multiple web pages
* Building a simple static website

## Project Features

* Homepage with links to individual recipes
* Separate recipe pages for different dishes
* Structured content using semantic HTML
* Easy navigation between pages

## Reflection

This project helped me develop a solid understanding of how websites are structured before introducing CSS and JavaScript. It strengthened my ability to organize content effectively and create clear navigation between pages.

Completing this project marks an important first step in my web development journey and provides a strong foundation for building more advanced and interactive web applications in the future.
